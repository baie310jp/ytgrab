# ytgrab
Download YouTube videos as MP3 or MP4 — auto-embeds title, artist, and artwork

## Installation
```zsh
cp -p src/videos.py.example src/videos.py
```

```zsh
uv sync
```

## Usage

### MP3
```zsh
uv run python src/main.py
```

or

```zsh
uv run python src/main.py mp3
```

### MP4
```zsh
uv run python src/main.py mp4
```
